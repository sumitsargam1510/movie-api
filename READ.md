# Movie Management REST API

A simple Spring Boot REST API for managing movies using in-memory storage.

## Features
- Add a movie
- Get movie by ID
- Input validation
- In-memory storage using ArrayList

## Tech Stack
- Java 17
- Spring Boot
- Maven

## API Endpoints

### Add Movie
POST /api/movies

### Get Movie
GET /api/movies/{id}

## Run Locally
mvn spring-boot:run
